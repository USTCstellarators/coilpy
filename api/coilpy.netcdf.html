coilpy.netcdf#

class coilpy.netcdf.Netcdf(filename, mmap=False, version=1, maskandscale=False)[source]#

Bases: object

Assembly datasets in a netcdf file into python classes. It takes the same arguments as scipy.io.netcdf.netcdf_file. Once parsed, you can access all the data directly via its key as class attributes.

Examples

Read a netcdf file:

data = Netcdf(‘path-to-file.nc’)

Access its attributes:

data.key

List all the keys:

print(list(data))
